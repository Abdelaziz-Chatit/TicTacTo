# Tic-Tac-Toe (Human vs Human)

This is a simple command-line implementation of the Tic-Tac-Toe game where two human players can play against each other.

## How to Play

1. Run the Python script `tic_tac_toe.py`.
2. Players take turns entering the row and column numbers to make their moves.
3. The game continues until one player wins or it's a tie.
4. Enjoy playing!

## Requirements

- Python 3.x
- NumPy library (if not installed, you can install it using `pip install numpy`)
