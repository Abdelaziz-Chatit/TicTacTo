# Tic-Tac-Toe with Random Move

This project implements a basic Tic-Tac-Toe game in Python, featuring a computer opponent that makes random moves.

## How to Play

1. Clone or download the repository to your local machine.
2. Run the `tic_tac_toe_random.py` file using a Python interpreter.
3. Follow the prompts to make your moves against the computer opponent.

## Features

- Interactive Tic-Tac-Toe gameplay.
- Computer opponent makes random moves.
- Simple and easy-to-understand code.

## Requirements

- Python 3.x
