# Tic-Tac-Toe with AI (Minimax Algorithm)

This project implements a Tic-Tac-Toe game in Python, featuring an AI opponent that uses the Minimax algorithm to make intelligent moves.

## How to Play

1. Clone or download the repository to your local machine.
2. Run the `tic_tac_toe_ai.py` file using a Python interpreter.
3. Follow the prompts to make your moves against the AI opponent.

## Features

- Interactive Tic-Tac-Toe gameplay.
- AI opponent uses the Minimax algorithm for intelligent moves.
- Simple and easy-to-understand code.

## Requirements

- Python 3.x
